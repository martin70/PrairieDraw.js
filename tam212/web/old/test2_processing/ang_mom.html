Angular Momentum in 3D

The angular momentum of a body describes its tendency to continue rotational motion, analogous to the way that linear momentum describes the tendency to continue translational motion. Angular momentum in 3D is a vector quantity \(\boldsymbol{H}\), while in 2D it is a scalar \(H\). The total angular momentum of several bodies is the sum of their individual angular momenta. Angular momentum changes due to the action of moments on the body, as described by Newton’s moment equation.

The angular momentum of a body depends on the reference point. We write \(\boldsymbol{H}_C\) to indicate that this angular momentum is with respect to the point \(C\).

Definition of angular momentum in 3D

Point masses

(Video explanation)

The angular momentum of a point mass \(P\) about some origin \(O\) is

\[\label{eqn:ang_mom_3d_point_mass} \boldsymbol{H}_O = \boldsymbol{r}_{OP} \times m \boldsymbol{v} = \boldsymbol{r}_{OP} \times \boldsymbol{p},\]

where \(\boldsymbol{r}_{OP}\) is the position vector of the point mass, \(m\) is the mass, \(\boldsymbol{v}\) is the velocity, and \(\boldsymbol{p} = m \boldsymbol{v}\) is the linear momentum.

Rigid bodies

(Video explanation)

The angular momentum of a rigid body about its center of mass \(C\) depends on its angular velocity \(\omega\) and moment of inertia matrix \(\boldsymbol{I}_C\), with

\[\label{eqn:ang_mom_3d_rigid_body} \boldsymbol{H}_C = \boldsymbol{I}_C \boldsymbol{\omega}.\]

Properties of angular momentum

Dependence on reference point

The angular momentum of a body depends on the reference point. We write \(\boldsymbol{H}_C\) to indicate that this angular momentum is with respect to the point \(C\).

Consider the example below, with a point (the black dot) moving in uniform circular motion. The red arrows show the angular momentum about nine different reference points. We see that \(H_e\) has a constant value in the counter-clockwise direction, while the other angular momenta vary over time and even reverse direction. This is despite the fact that the point is always moving with a constant angular velocity about the center \(e\). Note that the largest angular momentum is about the furthest-away point.

Another example is shown below, this time with the point moving in a square with variable speed. Again we note that the angular momentum is larger about far-away points. Also note that many of these angular momenta are equal, as the projection of the linear momentum onto the perpendicular direction gives the same value.

Adding angular momenta

(Video explanation)

If we have two bodies, whether connected or not, then the total angular momentum is the sum

\[\label{eqn:ang_mom_addition} \boldsymbol{H}_O^{\rm tot} = \boldsymbol{H}_O^1 + \boldsymbol{H}_O^2\]

of the angular momenta \(\boldsymbol{H}_O^1\) and \(\boldsymbol{H}_O^2\) of bodies 1 and 2.

Angular momenta can only be added together if they are both about the same reference point. It makes no sense to write \(\boldsymbol{H}_O^1 + \boldsymbol{H}_C^2\), for example.
